# Page WAJAAIS - Structure

## Structure de base

La page utilise trois éléments principaux :
- **html** - Le document complet
- **head** - Les informations de la page
- **body** - Le contenu visible

## En-tête et Navigation

L'en-tête contient un menu de navigation organisé avec :
- **header** - Zone d'en-tête
- **nav** - Section de navigation
- **ul** - Liste des liens
- **li** - Chaque élément du menu
- **a** - Les liens cliquables

## Contenu Principal

Le contenu est structuré avec :
- **main** - Zone de contenu principal
- **section** - Sections thématiques
- **h1** - Titre principal (le plus important)
- **h2** - Titres de sections
- **h3** - Sous-titres
- **p** - Paragraphes de texte
- **div** - Blocs de contenu
- **span** - Portions de texte en ligne

## Mise en page avec CSS Grid

La disposition en grille utilise :
- **display grid** - Active le système de grille
- **grid-template-columns** - Définit le nombre et la taille des colonnes
- **grid-template-rows** - Définit le nombre et la taille des lignes
- **gap** - Espace entre les éléments
